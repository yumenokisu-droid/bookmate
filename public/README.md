# BOOKMATE v2.8 - Seed Knowledge Cards + Dynamic Book Engine

반영 내용
- 대표도서 10종 사전 지식 카드 추가
- 대표도서는 검색/분석 대기 없이 완성된 지식 카드에서 바로 불러오기
- 지식 카드 항목: 책 정보, 핵심 소개, 줄거리 흐름, 주요 인물, 주제, 독서토론 질문, 생각정리 질문, 독서모임 질문, 유사 추천도서, 주의점
- 신규/신간 도서는 기존 Book Validator + Google Books + Gemini 분석 흐름 유지
- 정보 부족 시 지어내지 않고 추가 확인 필요로 표시하도록 지식 엔진 프롬프트 강화
